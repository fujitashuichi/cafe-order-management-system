# 📋 プロジェクト名：(仮) Virtual Case Cafe Plus
[以前のポートフォリオ](https://virtual-case-cafe.vercel.app/)をベースに、React と EF Core を用いて「リアルタイムな注文体験」と「バックエンドでのデータ管理」を実現するフルスタック・ポートフォリオ。

## このポートフォリオで解決する課題
* 店舗スタッフの願い: メニュー管理や注文確認を効率化したい
* 一般客の願い: スマホからスムーズに注文したい

## 主要機能

[ ] カテゴリー別メニュー表示（EF Core リレーション活用）

[ ] ショッピングカート機能（React State管理）

[ ] 注文送信機能（API連携 / DB保存）

[ ] 簡易管理画面（注文履歴の閲覧）

## 技術スタック

Frontend: Vite + React (TypeScript)

Backend: ASP.NET Core (Minimal API)

DB: SQLite + Entity Framework Core

Styling: Tailwind CSS (または以前のCSSを移植)
